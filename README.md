# Prophecy Tablet Archive

A website where you can play the Prophecy Tablet puzzles in Amphoreus from the game *Honkai: Star Rail*.

## How to Play

Drag and arrange the pieces on the right so they form the exact shape shown on the left. When pieces overlap, their areas behave differently: if an even number of pieces overlap in a spot, that area disappears, but if an odd number overlap, it stays visible. Keep adjusting and layering pieces until the combined shape perfectly matches the target shape on the left.
